# About

### Description
Raining-Sun is a web application that returns to the user the climate conditions on his/her location or whichever city is specified. 

### Features
* Weather
* Temperature

### Techonologies
* HTML 5
* CSS 3
* JavaScript
* React
* Other minor libraries

#### External API's
* https://openweathermap.org/

### Requirements
* NodeJS
* NPM package manager
